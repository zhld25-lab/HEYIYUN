# 数据模型

## Project

项目主数据，字段包括项目编号、名称、类型、电压等级、建设单位、项目经理、状态、计划周期、合同金额、目标成本、实际成本、已收款、已付款、产值进度、收款进度、成本比、利润率、风险等级、资料完整率。

## Contract

合同数据，字段包括合同编号、合同名称、合同类型、所属项目、甲方、乙方、合同金额、结算金额、发票金额、已收款、已付款、应收款、应付款、合同状态、审批状态、归档状态。

## Cost

成本数据，字段包括成本编号、成本类型、所属项目、关联合同、供应商/分包商、金额、发生日期、经办人、审批状态、发票状态、付款状态。

## Material

材料数据，字段包括材料编码、材料名称、规格型号、单位、数量、预算单价、采购单价、总金额、所属项目、供应商、采购合同、到货状态、入库状态、领用状态、库存数量、是否超价、是否缺货。

## Equipment

设备周材数据，字段包括设备编号、设备名称、规格型号、所属单位、当前项目、当前状态、使用开始日期、使用结束日期、租赁单价、租赁金额、维修状态、保养状态、责任人。

## Workflow

审批数据，字段包括流程编号、流程类型、标题、所属项目、发起人、当前节点、当前处理人、发起时间、到达时间、是否超时、审批状态、审批意见、流程轨迹。

## Risk

风险数据，字段包括风险编号、风险类型、风险等级、所属项目、触发原因、风险指标、责任人、截止时间、建议动作、当前状态、是否闭环。

## Document

资料数据，字段包括资料编号、资料名称、资料类型、所属项目、上传人、上传时间、审核状态、是否必传、是否缺失、版本号、附件路径。
