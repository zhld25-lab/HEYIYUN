# 后续升级计划

## Streamlit Demo 的局限

当前版本使用 mock 数据和 session state 模拟权限，适合演示业务方案和原型验证，但不适合作为真实生产系统直接使用。限制包括无真实数据库、无真实登录、无文件存储、无并发审批引擎、无细粒度权限和无审计级日志。

## Vue + FastAPI + PostgreSQL 方案

- 前端：Vue 3、TypeScript、Element Plus 或 Ant Design Vue。
- 后端：FastAPI，按项目、合同、成本、采购、资金、审批、资料、风险拆分业务模块。
- 数据库：PostgreSQL，设计项目主表、合同表、成本表、材料表、设备表、资金流水表、流程实例表、资料表和风险表。
- 文件：对象存储或企业网盘，保存资料附件和现场照片。
- 权限：RBAC + 数据权限，支持公司、部门、项目维度授权。

## 真实企业系统增强

- 真实登录、单点登录和组织架构。
- 数据库持久化和审计日志。
- 文件上传、资料版本和电子签章。
- 审批流引擎和流程配置器。
- 移动端、小程序和现场扫码填报。
- 与财务软件、OA、采购平台和电子发票系统集成。
- 项目经营分析、预算控制和智能风险预警。
